# ESG-site | Портфолио AI-креатора

---

**Название проекта:** ESG-site — сайт-портфолио AI-креатора Дмитрия Лазарева

**Задача:** Создать сайт, на котором потенциальные клиенты могут за 30 секунд оценить качество работ, понять подход и напрямую выйти на автора. Единственный сайт, который объединяет кейсы по ESG-коммуникации, образованию и личному бренду в одном месте.

**Решение:** Страница-портфолио с интерактивными видео. Один ролик воспроизводится прямо на странице (hover — без звука, click — со звуком), остальные показаны как превью-скриншоты для скорости загрузки. Каждый проект сопровождается описанием и тегами (сценарий, AI-аватар, монтаж). К шаблону прилагаются готовые шаблоны откликов для клиентов.

**Пользователь:**
| Кому подходит | Что получает |
|---|---|
| Топ-менеджмент и акционеры | Видит, как выглядит AI-видео для коммуникаций |
| Отделы маркетинга и коммуникаций | Оценивает, как упаковать ESG-повестку в ролики |
| EdTech-проекты | Видит, как обратиться к родителям через видео |
| Эксперты и блогеры | Видит, как формировать личный бренд через AI-аватар |

---

## Что на сайте

- Интерактивное видео (0708.mp4) —.hover-воспроизведение, клик включает звук
- 3 превью-скриншота из видео с описанием проектов
- Анимированный фон (тёплые тона)
- Полный перечень инструментов
- Шаблоны откликов для клиентов (уникальный + для топ-менеджмента)
